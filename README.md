# mugenCanvas

アニメーション制作用デスクトップアプリ。Rust + eframe (egui + wgpu + winit) で構築。

## 操作手順

### 起動 / プロジェクト管理

| 操作 | 方法 |
|------|------|
| 新規プロジェクト | トップバー **NEW** ボタン、または `Ctrl+N` |
| プロジェクトを開く | トップバー **OPEN** ボタン、または `Ctrl+O` |
| 上書き保存 | トップバー **SAVE** ボタン、または `Ctrl+S` |

プロジェクトファイルは `.mugencanvas` 形式（JSON + 画像データ）。初回保存時にファイル名と保存先を選ぶ。

---

### ツール選択

左パネルのボタンまたはキーボードショートカットでツールを切り替える。

| ツール | キー | 説明 |
|--------|------|------|
| ブラシ | `B` | アンチエイリアス付きブラシ描画。筆圧対応 |
| 消しゴム | `E` | 消しゴム（ブラシとは独立した設定） |
| 丸ペン | `P` | 筆圧なし・くっきりエッジの丸ペン（色塗り向け） |
| 塗りつぶし | `G` | クリックした色領域を BFS 塗りつぶし |
| 手のひら | `H` | キャンバスをパン |
| ズーム | `Z` | クリックでズームイン / `Alt+クリック` でズームアウト |
| 回転 | `R` | キャンバスを回転（ドラッグで回転） |
| カメラ | `C` | カメラ枠の移動・リサイズ、キーフレーム操作 |

`Space` を押しながら操作すると、どのツール中でも一時的に手のひらツールに切り替わる。

---

### キャンバス操作

| 操作 | 方法 |
|------|------|
| パン (移動) | `Space + ドラッグ`、または手のひらツールでドラッグ、またはホイールクリック+ドラッグ |
| ズーム | `Ctrl + ホイール` または ズームツールでクリック |
| 回転 | 回転ツール (`R`) を選択してキャンバス内を左右にドラッグ |
| 回転リセット | `Ctrl+R` |
| 微回転 | `Ctrl+E` (右) / `Ctrl+Q` (左) で 20° ずつ |

---

### ブラシ / 丸ペン / 消しゴム設定

左パネル下部のカラーパネルで設定する。ブラシ・丸ペン・消しゴムはそれぞれ独立した設定を保持する。設定は次回起動時も引き継がれる。

| 設定項目 | ブラシ | 丸ペン | 消しゴム |
|----------|--------|--------|---------|
| カラー | ✓ | ✓ | — |
| サイズ | ✓ | ✓ | ✓ |
| 不透明度 | ✓ | ✓ | — |
| アンチエイリアス | on (固定) | off (固定・くっきり) | — |
| 筆圧感度 | ✓ (スタイラス使用時) | なし | — |

**丸ペン**はアンチエイリアスなし・筆圧なしのくっきりとした丸ブラシ。主に色塗り用途に使用する。

---

### レイヤー操作

右パネルでレイヤーを管理する。

| 操作 | 方法 |
|------|------|
| レイヤー追加 | **+** ボタン |
| レイヤー削除 | 選択して **-** ボタン（1枚以上必須） |
| レイヤー名変更 | レイヤー名をクリックして編集 |
| 表示/非表示 切替 | 目のアイコンをクリック |
| ロック/ロック解除 | 鍵アイコンをクリック（ロック中は描画不可） |
| クリッピング切替 | **⌐** ボタンをクリック（青 = ON） |
| 並び替え | ↑ / ↓ ボタンで移動 |
| レイヤー選択 | 一覧でクリックして選択 |

#### クリッピングについて

クリッピングを ON にしたレイヤーは、直下のレイヤーの描画範囲だけに表示が制限される（クリップスタジオペイントのクリッピング機能と同等）。線画レイヤーの上に色塗りレイヤーを追加してクリッピングすると、線画の外にはみ出さずに塗ることができる。

---

### タイムライン / コマ管理

画面下部のタイムラインでフレームを管理する。

| 操作 | 方法 |
|------|------|
| フレーム移動 | タイムラインをクリック、または `←` / `→` キー |
| 再生 / 停止 | `Enter` キー |
| コマ割り当て (hold) | タイムラインのセルを右クリック → **コマ割り当て** |
| コマ解除 | コマのセルを右クリック → **コマ解除** |
| コマ切替 (キー) | `.` キー でカレントフレームのホールドを on/off |

#### コマ (hold) について

- ◯ マーク: そのフレームに描画データがある
- · マーク: 直前の ◯ フレームを参照している (hold フレーム)
- hold フレームへの描画は自動的にブロックされる

---

### フレームのコピー / ペースト

選択中のレイヤーの現在フレームをコピー・ペーストする。

| 操作 | キー |
|------|------|
| フレームコピー | `Ctrl+C` |
| フレームペースト | `Ctrl+V` |

---

### オニオンスキン

タイムラインパネル内のオニオンスキン設定で on/off と透明度を調整できる。

- 前フレームは青色で重ねて表示
- 後フレームは赤色で重ねて表示

---

### グリッド表示

| 操作 | 方法 |
|------|------|
| グリッド on/off | `Ctrl+G` またはトップバーの **GRID** ボタン |
| グリッドサイズ変更 | トップバーの数値フィールドで設定 |

---

### カメラ枠

カメラツール (`C`) を選択して操作する。

| 操作 | 方法 |
|------|------|
| カメラ枠を移動 | 枠内をドラッグ |
| カメラ枠をリサイズ | 四隅のハンドルをドラッグ |
| キーフレーム追加 | タイムラインのカメラ行 `+` ボタン、またはセルを右クリック → **キーフレーム追加** |
| キーフレーム削除 | キーフレームセルを右クリック → **キーフレーム削除** |
| イージング変更 | キーフレームセルを右クリック → **イージング** から選択 |

#### イージング種別

| 種別 | 説明 | ドットカラー |
|------|------|------|
| リニア | 一定速度 (デフォルト) | 金色 |
| イーズイン | ゆっくり始まり速く終わる | 青 |
| イーズアウト | 速く始まりゆっくり終わる | 緑 |
| イーズインアウト | ゆっくり始まりゆっくり終わる (S字) | 紫 |
| ホールド | 次のキーフレームで瞬間移動 | 赤 |

カメラ枠の外枠は赤い線で表示される。

---

### 書き出し

| 形式 | 操作 |
|------|------|
| 現在フレームを PNG | トップバー **PNG** ボタン |
| 連番 PNG | トップバー **IMAGES** ボタン → フレーム範囲指定 → フォルダ選択 |
| アニメーション GIF | トップバー **GIF** ボタン → フレーム範囲指定 → ファイル名を指定 |
| MP4 動画 | トップバー **動画カメラ** ボタン → フレーム範囲指定 → ファイル名を指定 |

書き出しはカメラ枠内の領域を使用する。GIF は最大幅 960px でリサイズ。MP4 はアプリに内蔵した H.264 エンコーダーを使用するため、外部ツールのインストールは不要。

---

### 自動アップデート

起動時にネットワーク接続がある場合、GitHub リリースから最新版を自動確認する。新しいバージョンが見つかると更新ダイアログが表示される。

| 操作 | 方法 |
|------|------|
| 起動時の自動確認 | 新しいバージョンがあれば起動時にダイアログ表示 |
| 手動確認 | トップバーの **↑** ボタン |

**↑** ボタンは更新が利用可能なとき緑色に変わる。「今すぐ更新」をクリックするとバイナリをダウンロードして自動的に置き換え、新バージョンで再起動する。オフライン時はチェックをスキップして通常起動する。

---

### アンドゥ / リドゥ

| 操作 | 方法 |
|------|------|
| アンドゥ | `Ctrl+Z` またはトップバー **↩** ボタン |
| リドゥ | `Ctrl+Y` / `Ctrl+Shift+Z` またはトップバー **↪** ボタン |

20 段まで履歴を保持する。

## 技術スタック

| 役割 | 技術 |
|------|------|
| ウィンドウ・入力 | winit (eframe 経由) |
| GPU レンダリング | wgpu (eframe 経由) |
| UI フレームワーク | egui 0.29 |
| キャンバス描画 | CPU タイル合成 → egui TextureHandle |
| ファイルダイアログ | rfd 0.15 |
| GIF エンコード | gif + rayon (並列 NeuQuant) |
| 保存フォーマット | serde_json + base64 PNG タイル |

## セットアップ / 実行

Rust toolchain (stable) が必要。Node.js / npm は不要。

```bash
cargo run            # 開発実行 (プロジェクトルートから)
cargo build --release  # リリースビルド
```

## ディレクトリ構成

```
src/
  main.rs          # エントリーポイント (eframe::run_native)
  app.rs           # MugenCanvasApp (eframe::App 実装、入力処理)
  model.rs         # データモデル (serde camelCase)
  state.rs         # AppState
  undo.rs          # UndoManager (スナップショット方式)
  canvas/
    mod.rs         # CanvasState, タイル管理, CPU 合成, エクスポート
    brush.rs       # draw_segment, flood_fill
    viewport.rs    # 座標変換 (screen ↔ world)
    camera.rs      # カメラキーフレーム線形補間
  ui/
    topbar.rs      # トップバー (メニュー・操作)
    toolbar.rs     # ツール選択
    color_panel.rs # カラー・ブラシ設定
    layers_panel.rs  # レイヤー一覧
    timeline.rs    # タイムライン
  io.rs            # 保存・読み込み・PNG/GIF 書き出し
```
