## 技術メモ

### 保存フォーマット (現行 MVP)
```json
{
  "app": "mugenCanvas",
  "version": "0.1.0",
  "settings": { "width": 1920, "height": 1080, "fps": 24, "totalFrames": 60 },
  "layers": [ ... ],
  "cameraTrack": { ... },
  "drawings": {
    "layer-bg:1":  "data:image/png;base64,...",
    "layer-1:3":   "data:image/png;base64,..."
  }
}
```

### OffscreenCanvas サイズ
- プロジェクト設定 (width × height) に一致。デフォルト 1920×1080。
- プロジェクト読み込み時に `Renderer.setCanvasSize()` で自動リセット。

### コマ (hold) データモデル
- `AnimationFrame.drawingId !== null` → 描画あり (◯)
- `AnimationFrame.holdSource !== null` → hold フレーム (·)、描画ブロック
- `Renderer.resolveLayerFrame()` で hold チェーンを解決して描画元フレームを返す
